# 公众号写作详细规则与示例

本文档是 `公众号爆文写作` skill的详细规则参考，写正文时加载使用。

**⚠️ 写作前必读**：`禁止规则清单.md`（同目录）—— 明确"不能做什么"比"要做什么"更重要。

---

## 一、选题3大底层逻辑（来自25讲第07讲）

选题决定了文章的天花板。好选题 + 一般写法 > 差选题 + 极好写法。

### 1.1 潜在读者数量

选题覆盖的人群有多大，决定了你的阅读量天花板。

- **大众话题**（赚钱、省钱、职场、婚姻）→ 潜在读者几亿
- **垂直话题**（AI赚钱、自媒体变现）→ 潜在读者几百万
- **窄众话题**（某款工具使用教程）→ 潜在读者几万

**原则**：在垂直赛道里，选大众痛点的选题。比如"AI赚钱"是垂直赛道，但"普通人怎么用AI赚钱"是大众痛点。

### 1.2 痛点程度

选题戳得越痛，打开率越高。

- **一级痛点**：生存、安全、赚钱、省钱 → 必须解决，不得不看
- **二级痛点**：效率、成长、关系 → 想解决，会关注
- **三级痛点**：审美、兴趣、好奇 → 可看可不看

**原则**：优先选一级痛点。如果选二级三级，必须用"展示回报"或"颠覆认知"来补。

### 1.3 社交原力

读者看完会不会转发？转发动机是什么？

- **身份表达**：转发这篇文章 = 我是这样的人
- **社交货币**：转发 = 我掌握了一个别人不知道的信息
- **情绪宣泄**：转发 = 替我说了想说的话
- **利他心理**：转发 = 帮朋友解决问题

**选题时问自己**：读者看完这篇文章，会为了什么理由转发？

### 1.4 选题实战流程

1. 先想3-5个选题方向
2. 用3个底层逻辑打分（1-5分）
3. 3个维度都≥3分的，优先写
4. 如果某个维度特别高（5分），其他维度2分也可以写

## 二、标题8大技巧详解（来自25讲第08讲）

### 2.1 激发好奇

制造信息缺口，让读者觉得"我必须知道答案"。

示例：
- "一个很笨，但能让你变有钱的方法"
- "体制内裸辞……才发现，普通人赚不到钱，原因就1个！"

技巧：用"笨""1个""秘密"等词暗示信息缺口。

### 2.2 激发认同

替读者说出他们心里的话，让人觉得"对对对就是这样"。

示例：
- "如果你的存款有二三十万，请牢牢记住这3条保富法则"
- "普通人赚钱太难了，但花钱一定要谨慎"

技巧：用"你"拉近距离，用普遍感受引发共鸣。

### 2.3 展示负面

警示风险，让人不得不看。"不看可能吃亏"。

示例：
- "手里有存款的普通人，千万别做这3件事"
- "你的钱正在被这样慢慢掏空"

技巧：用"千万别""正在被"制造紧迫感。

### 2.4 展示回报

告诉读者看完能得到什么，直接给预期。

示例：
- "掌握这3个思维，你的收入至少翻一倍"
- "普通人用AI赚钱，最快的方法就这一条"

技巧：回报要具体、可信，不要夸大到不可信。

### 2.5 展示新闻

借热点/新事件的时效性，搭便车。

示例：
- "XXX裸辞做AI博主，单价翻了7倍：AI时代最值钱的是什么"

技巧：新闻只是引子，核心要回到你的观点/方法。

### 2.6 呈现对比

用反差制造冲突感，打破预期。

示例：
- "月薪3000的她，3年后年入百万：做对了什么？"
- "同样是做自媒体，有人月入10万有人颗粒无收"

技巧：对比的两个状态差距越大越好。

### 2.7 颠覆认知

打破常识，引发好奇。

示例：
- "10倍比2倍更容易：为什么你越努力越赚不到钱"
- "存钱越多的人，可能越不会变有钱"

技巧：颠覆不是胡说，是换一个角度说真话。

### 2.8 事实+观点

前半句说事实，后半句给观点。案例文标配。

示例：
- "前大厂产品经理裸辞，做AI博主，单价从900涨到6500：AI时代最值钱的，是翻译官"

技巧：事实要有数字，观点要有力。中间用冒号分隔。

### 2.9 标题组合技

8种技巧可以组合使用，威力翻倍：
- 事实+观点 + 颠覆认知 = 最强组合
- 激发好奇 + 展示回报 = 学习类标题
- 展示负面 + 激发认同 = 避坑类标题

## 三、8种爆文框架详解（来自25讲第09讲）

### 3.1 干货文框架：是什么→为什么→怎么做

最实用的框架，交付感最强。

```
【开头】痛点引入：你有没有遇到这个问题？
【01】是什么——说清楚这个方法/概念
【02】为什么——为什么这个方法有效
【03】怎么做——具体步骤，读者拿去就能用
【结尾】强化价值 + 互动
```

适用：教方法、教技能、教思路的文章。

### 3.2 观点文框架：总→分→总

最有力量感的框架，说服力强。

```
【开头】直抛核心观点
【01】分论点1 + 案例
【02】分论点2 + 案例
【03】分论点3 + 案例
【结尾】重申核心观点 + 共鸣
```

适用：表达认知、颠覆常识、给判断的文章。

### 3.3 清单体框架：N个要点并列

最易读的框架，适合碎片化阅读。

```
【开头】为什么这N个点很重要
【01】要点1：观点→解释→案例
【02】要点2：观点→解释→案例
【03】要点3：观点→解释→案例
...
【结尾】总结 + 互动
```

适用：多条建议、多个方法、多个认知的文章。

### 3.4 裂变法框架：痛点→方法→效果→深层方法→更大效果

递进式交付，一环扣一环。

```
【开头】大痛点引入
【01】初步方法——先解决表面问题
【02】效果展示——用了之后的变化
【03】更深层方法——为什么还有更优解
【结尾】更大效果 + 引导行动
```

适用：方法递进、层层深入的文章。销售软文常用。

### 3.5 人物稿框架：发现机会→怎么赚钱→普通人怎么学

案例文标配框架，Vicky手改终稿验证。

```
【开头】人物背景→关键事件→抛出钩子
【01】她是怎么发现机会的
【02】她现在怎么赚钱的（具体故事+数字）
【03】普通人怎么学（可复制的方法）
【结尾】呼应主题 + 共勉 + 互动提问
```

适用：人物案例、赚钱故事、转型故事。

### 3.6 同维度并列：A/B/C并列论证同一观点

多个案例从同一维度论证，制造"大势所趋"感。

```
【开头】核心观点
【01】案例A——正面佐证
【02】案例B——正面佐证
【03】案例C——正面佐证
【结尾】所以结论是……
```

适用：论证趋势、论证普遍现象。

### 3.7 逻辑递进：A→B→C层层推进

后一个论点基于前一个，层层递进，不可反驳。

```
【开头】核心观点
【01】第一步逻辑
【02】基于01的进一步推论
【03】基于02的最终结论
【结尾】强化结论
```

适用：认知类文章、说服性文章。

### 3.8 不同维度并列：从不同角度论证同一观点

从心理、行为、结果等不同维度论证，面面俱到。

```
【开头】核心观点
【01】维度1：心理层面
【02】维度2：行为层面
【03】维度3：结果层面
【结尾】多维论证后收束
```

适用：复杂观点、需要多角度说服的文章。

## 四、开头5技巧详解（来自25讲第11讲）

### 4.1 提问式

用问题勾住读者，制造悬念。

示例：
> "你有没有想过，为什么有的人越忙越穷，有的人越闲越有钱？"

技巧：问题要和读者利益直接相关，不要问空泛的大问题。

### 4.2 反常识事实/观点

打破读者预期，引发好奇。

示例：
> "对大多数普通人而言，二三十万算不上巨款，却是一个人生关键拐点。"

技巧：反常识不是胡说，是"看起来矛盾但说通了"。

### 4.3 猎奇故事

用一个奇特的人/事开头。

示例：
> "门东东大学毕业之后，成了体制内一名大学老师，周围人都特别羡慕她。直到有一天，一位马上要退休的前辈说：年轻人们好好干，再过40年，你们也能躺平了。"

技巧：故事要有转折，有画面感，让人想看下去。

### 4.4 新闻热点

借热点引入话题。

示例：
> "最近AI行业又炸了，XXX公司发布了……"

技巧：热点只是引子，1-2句带过，快速回到你的观点。不要花半篇讲热点本身。

### 4.5 对我有用

直接告诉读者"这跟你的利益相关"。

示例：
> "如果你也想用AI赚钱，但不知道从哪入手，这篇文章就是写给你的。"

技巧：直接承诺价值，让读者觉得"这值得我花5分钟"。

## 五、结尾5技巧详解（来自25讲第11讲）

### 5.1 金句共鸣

用一句有力量的话收束，让读者想截图/转发。

示例：
> "一生只用富一次。"
> "赚钱已经足够难了，花钱的时候一定要谨慎又谨慎。"

技巧：金句不是排比，不是对仗，是"一句说到心坎里"。

### 5.2 强化主题

再点一次核心观点，加深印象。

示例：
> "说到底，只会赚钱，从来都不等于能真正变有钱。"

技巧：不要原样重复标题，换一个角度说同一件事。

### 5.3 强化价值

强调读者能得到什么，给出行动指引。

示例：
> "记住这3条法则，你的存款就是你的底气。"

技巧：给读者一个"带走的东西"。

### 5.4 制造话题

抛出一个开放性问题引发讨论。

示例：
> "❓️你的赚钱模式，是单一的，还是多元的？"

技巧：问题要和文章主题直接相关，且容易回答。

### 5.5 呼吁行动

引导点赞/转发/评论/购买。

示例：
> "如果对你有点启发，欢迎【点赞】或【转发】给有需要的朋友，一起分享。"
> "❓️看完这篇，你打算怎么行动？欢迎【评论区】一起聊聊~"

技巧：行动要具体，不要说"欢迎关注"，要说"欢迎【评论区】一起聊聊"。

## 六、素材方法论详解（来自25讲第10讲）

### 6.1 素材12大渠道

| 渠道 | 搜什么 | 独特价值 |
|------|--------|---------|
| 个人经历 | 你的决策、转折、教训 | 不可复制，最真实 |
| 身边人故事 | 亲戚朋友的真实故事 | 接地气，读者容易代入 |
| 微信 | 朋友圈/群聊/公众号留言 | 真实社交场景 |
| 微博 | 热搜/评论区故事 | 时效性强，话题广 |
| 知乎 | 高赞回答/真实经历 | 深度内容，质量高 |
| 小红书 | 生活方式/消费决策 | 消费者视角，年轻女性 |
| B站 | 长视频深度内容/弹幕 | 深度+情绪，弹幕是洞察 |
| 百度百科 | 基础事实核查 | 权威性背书 |
| 垂直网站 | 行业论坛/专业社区 | 专业深度 |
| 视频平台 | 抖音/快手/视频号 | 评论区是金矿 |
| 影剧纪录片播客 | 故事素材/金句 | 故事性强，金句多 |
| 书籍课程 | 系统性知识/权威观点 | 体系完整，权威性高 |

### 6.2 素材5大搜索技巧

1. **多一点耐心**：翻3页以上，好素材在后面。第一页的内容大家都看过了，没有稀缺性。
2. **多关键词搜索**：同一个话题换3-5组关键词搜。搜"省钱"同时搜"攒钱""存钱""不花钱""节流"。
3. **延伸关键词**：搜A的时候，也搜A的上下游话题。搜"AI赚钱"的同时搜"AI接单""AI副业""AI工具变现"。
4. **多渠道交叉**：微博搜完去知乎搜，知乎搜完去小红书搜。不同平台用户画像不同，素材角度也不同。
5. **多形式素材**：文字+视频+播客+评论，不同形式素材互补。一段视频里的真实表达，比文字更有感染力。

### 6.3 素材日常积累

- 建一个灵感库/素材库（备忘录/飞书/Notion）
- 看到好故事、好金句、好数据，随手记
- 标注来源和可用场景
- 每周花30分钟整理一次

## 七、遣词造句规则

### 7.1 必须做的事

| 规则 | 说明 | 示例 |
|------|------|------|
| 主语用"你""我" | 对话感，一对一说话 | "你本来就是一个普通人，赚钱太难了" |
| 用逻辑路标词 | 推进逻辑，让读者跟上 | "说到底""为什么这么说""举个例子" |
| 用连接词串逻辑 | 逻辑链清晰 | "其实"→"但是"→"所以"→"因为" |
| 用动词驱动 | 动作感，给力 | "抓住""记住""守住""避免""一定要" |
| 给确定性建议 | 敢下判断，不模糊 | "一定要""绝对不要""千万不要" |
| 口语化填充词 | 自然出现，不过度 | "左右""大概""吧""呢" |
| 数字制造可信度 | 具体数字替代空泛描述 | "二三十万""300万粉丝""100万""600万左右" |

### 7.2 禁止做的事

| 禁止 | 原因 | 替代方案 |
|------|------|---------|
| "综上所述""值得注意的是""基于此""由此可见" | 书面语，公众号不需要 | "说到底""说白了""其实" |
| "不是……而是……""真正的……是/不是……" | AI味句式 | 直接说事实 |
| 用引号加重词 | AI痕迹 | 不加引号，直接说 |
| 排比句、对仗句 | AI痕迹，假 | 用长短交替的自然节奏 |
| "你是不是也……" | 八股煽情 | 直接说问题 |
| 复杂长难句 | 读者读不下去 | 一句一个意思，30字以内 |
| "可能""也许""我觉得" | 模糊，没态度 | "一定""绝对""千万不要" |
| "人们""大家" | 抽象，没对话感 | "你""普通人""对大多数人来说" |
| 形容词堆砌 | 空泛 | 用具体数字+动词 |

## 八、案例列举方法

### 8.1 四种案例用法

**方法1：真实人物案例**（93%的文章使用）

用真实人名+真实经历，可信度极高。

示例：
> 门东东大学毕业之后，成了体制内一名大学老师，周围人都特别羡慕她，吃皇粮，稳定，饭碗有着落了。直到有一天，一位马上要退休的前辈说：年轻人们好好干，再过40年，你们也能躺平了。40年？一句话让她惊出一身冷汗，当天，她就决定裸辞了。

**方法2：自身经历案例**（87%的文章使用）

用"我"的真实决策/经历说明观点，比纯理论说服力强10倍。

示例：
> 我懂新媒体，懂公众号运营，但是如果一个300万粉丝的账号，想100万卖给我，我不会买。因为我会评估，如果全赔了我能不能输得起。

**方法3：多个并列案例堆叠**（67%的文章使用）

3个以上并列案例制造"大势所趋"感，读者自动对号入座。

示例：
> 有人抓住了淘宝，有人抓住了买房，有人抓住了公众号，有人抓住了比特币，有人抓住了抖音，有人抓住了起飞的股票，有人抓住了AI，有人抓住了知识付费……

**方法4：反面案例+正面建议**（高频使用）

先说"有人这么做赔了"，再说"你应该这么做"。先制造恐惧，再给出安全感。

示例：
> 有的人把钱都借给别人，要不回来；有的人好不容易攒到20万，结果开个饭店全赔了；有的人赚到一些钱之后买股票赔光了……所以，对于普通人来说，赚钱已经足够难了，花钱的时候一定要谨慎又谨慎。

### 8.2 案例使用禁忌

- 不要编造案例，用真实可查的
- 不要案例堆砌过多，一个观点1~2个案例足够
- 案例必须和观点直接相关，不跑题
- 案例中的数字必须具体，不用"很多""一些"等模糊词

## 九、谋篇布局详细规则

### 9.1 开头写法（按文章类型）

**观点文/干货文开头**：
- 200字内抛出核心观点
- 不绕弯子，不铺垫
- 可用反常识观点直击
- 示例："对大多数普通人而言，二三十万算不上巨款，却是一个人生关键拐点。"

**案例文开头**：
- 先交代人物背景（谁+做什么+什么处境）
- 有"逐步发现"的过程
- 示例："门东东大学毕业之后，成了体制内一名大学老师……直到有一天，一位前辈说……"

**IP长文开头**：
- 交代"我为什么写"
- 真诚最重要，不要端着
- 示例："想了很久要不要写这篇文章，最后还是写了。因为我觉得，有些弯路，我不走一遍你可能也会走。"

**销售软文开头**：
- 讲用户渴望的结果 或 代入感强的改变故事
- 让读者先看到"我也能这样"
- 示例："3个月前她还在上班，现在已经靠AI月入3万了。不是因为她比别人聪明，而是她做对了一件事。"

### 9.2 主体写法

- 严格用01/02/03数字分段，每段一个小标题
- 每段内部结构：观点→解释→案例→总结
- 每段只讲一个观点，不交叉
- 每段200~500字
- 金句独立成段（前后空行）
- 关键观点可用✅️❓️📌等符号标记

### 9.3 结尾写法

固定三件套：
1. 总结收束：一句话回扣核心观点
2. 行动呼吁："以上，有启发，请帮忙【点赞】或【转发】分享，感谢！"
3. 互动提问："❓️你的XX，是怎样的？❓️看完这篇，你打算怎么行动？欢迎【评论区】一起聊聊~"

## 十、核心写作原则——来自Vicky手改终稿的关键认知

**⚠️ 最重要的认知：公众号写作目标是内容深度和价值，不是过朱雀检测。**

从栗噔噔案例文v1→v2→终稿的迭代过程中，最关键的一条教训：

### 写完整句子，不要表演"碎句"

| ❌ 之前写的（模仿"人味"但失败了） | ✅ Vicky改的（真正说人话） |
|------|------|
| "她管自己叫'一人公司'。一个人，加一个AI舰队。"（碎句，信息不完整） | "她现在管自己叫'一人AI战舰'。一个人决策，多个AI助理执行。"（完整，信息充分） |
| "说个有意思的事儿。"（废话铺垫） | 直接说事，砍掉所有铺垫 |
| "到现在呢？一篇文章6500。视频5500。不到一年，单价翻了7倍多。月收入干到了上班时候的5倍。"（数据堆砌） | "从第一个商单900块，到现在的6500块，不到一年，单价飙了7倍多。"（开头只给核心数据，其余散落正文） |
| "第一条，做AI博主。公众号、小红书、视频号全做。"（电报式） | "先是自媒体，现在占她收入的一半。公众号、小红书、视频号。她的内容用户能看懂，能转化，品牌方排着队找她，最多的时候，6个甲方在轮流排号。"（完整叙述，有画面） |

**核心原则**：
- 正常人说完整句子。不是短才叫口语化。
- 一句话说清楚一件事。逻辑要有头有尾。
- 去掉所有"说个有意思的""按说""说白了"这种过渡废话。直接说事。
- 数据跟着事件自然散落，不要集中罗列。

## 十一、拆解方法论详解（来自25讲第13讲）

### 11.1 为什么拆解是提升写作最快的方法

看100篇文章不如拆解1篇文章。拆解让你从"读者视角"切换到"作者视角"，看到文章的骨架和手法。

### 11.2 选标4维度

1. **数据维度**：阅读量、点赞量远超平均的文章（至少2倍以上）
2. **热度维度**：最近1-3个月的热文，时效性参考价值大
3. **学习价值**：和你定位越接近的，越值得拆
4. **补短板**：你弱什么就拆什么。标题差就拆标题好的，开头差就拆开头好的

### 11.3 拆解4维度

**维度1：选题标题**
- 为什么这个选题能火？
- 标题用了8大技巧中的哪个？
- 如果换个标题，你会怎么写？

**维度2：脉络结构**
- 用的8种框架中的哪种？
- 段落怎么推进的？
- 哪些段落可以精简？

**维度3：爆款成因**
- 是什么让这篇文章被大量转发？情绪？价值？身份认同？
- 如果去掉某一段，还火吗？找出不可替代的部分
- 这篇文章的"社交原力"是什么？

**维度4：内容表达**
- 用了什么素材？从哪来的？
- 开头和结尾怎么写的？
- 金句怎么落的？
- 有没有让你"哇"的写法？学过来

### 11.4 拆解后应用5技巧

1. **找同定位多篇爆文**：一篇可能是运气，多篇就是规律
2. **提炼选题角度和标题套路**：发现你定位下的爆款选题方向
3. **总结框架结构和素材方法**：建立你自己的模板库
4. **重点学爆文开头和结尾**：开头和结尾是技巧最集中的地方
5. **合并创建灵感库**：持续积累，越积累越不缺灵感

## 十二、IP长文写法详解（来自25讲第18讲）

### 12.1 IP长文4大价值

1. **传播力**：真实故事最容易被转发
2. **信任**：读者了解你之后才会信任你
3. **完整画像**：一篇好IP文 > 100篇零散文章
4. **长期复用**：可以反复引用、置顶、发朋友圈

### 12.2 IP长文6要素

1. **为什么写**：开头交代动机，真诚最重要。"想了很久要不要写这篇文章"
2. **我是谁**：背景、经历、转折。不是简历，是故事
3. **重在过程讲变化**：不是展示结果，是讲怎么走到今天的。读者关心的是"怎么变的"，不是"你多厉害"
4. **用成就积累信任**：数据、案例、里程碑。不是炫耀，是让读者知道你说的话有根基
5. **有价值才有动机**：你的故事对读者有什么启发？读者能得到什么？
6. **真实是根基**：编的故事经不起时间检验。真实的才能长期用

### 12.3 IP长文3结构

```
01 我是谁
  - 背景：出生、成长、职业经历
  - 人生关键节点：转折、选择、困境
  - 怎么走到今天的：过程比结果重要

02 我在做什么
  - 现在的状态：做什么、为什么选这条路
  - 具体在做什么：日常、方法、成果
  - 选择了什么、放弃了什么

03 为什么做
  - 价值观：你相信什么
  - 长期主义：5年10年后的目标
  - 对读者的承诺：你做这件事跟他们有什么关系
```

### 12.4 IP长文注意事项

- 不要写成简历/成就报告
- 不要全是结果，要有过程中的挣扎和选择
- 不要端着，真诚暴露脆弱反而加分
- 字数可以长，3000-5000字都行
- 这篇文章会长期用，值得反复打磨

## 十三、销售软文写法详解（来自25讲第23讲）

### 13.1 销售软文5要点

1. **既要高阅读又要精准**：标题吸引泛流量，内容筛选精准用户。不是所有人买，但对的人一定要看到
2. **创造希望，不制造焦虑**：让读者看到可能性——"我也能这样"，而不是恐惧——"不学就完了"
3. **反复磨到90分**：销售软文差10分，转化差一倍。写完放一天，第二天再改一遍
4. **验证逻辑，避免自嗨**：让朋友读一遍，看他能不能被打动。如果他问"这跟我有什么关系"，说明开头要改
5. **暗中解决用户疑惑**：预判读者会问的问题（贵不贵？我能学会吗？真的有用吗？），在文中自然回答

### 13.2 销售软文4个选题方向

**方向1：讲用户渴望的结果**
先让他看到"我也能这样"，再告诉他怎么到达。

> "3个月从0到月入3万，她做对了什么？"

**方向2：讲代入感强的改变故事**
和自己越像的人的故事，越能打动。

> "一个40岁的宝妈，靠AI赚钱养家"

**方向3：讲解决方案**
直接给方法，自然过渡到"更系统的方案在课程里"。

> "普通人用AI赚钱的3个方法，第2个最容易被忽略"

**方向4：讲大痛点**
这个问题不解决，后果是什么。

> "还在靠死工资？3年后你会后悔的"

### 13.3 销售软文框架

```
【标题】渴望结果/改变故事/痛点
【开头】讲用户渴望的结果 或 代入感强的改变故事（先勾欲望）
【01】大痛点——为什么这个问题必须解决
【02】解决方案——你提供什么（自然引出产品/课程）
【03】效果证明——案例/数据/学员反馈
【结尾】行动呼吁 + 限时/限量（如适用）+ 互动
```

### 13.4 销售软文禁忌

- ❌ 一上来就卖——先给价值，再推荐产品
- ❌ 制造焦虑逼购买——创造希望才对
- ❌ 夸大效果/虚假承诺——说真话，长期信任比短期转化重要
- ❌ 产品植入太硬——用故事/案例自然过渡到产品
- ❌ 全篇都在卖——有价值内容占70%+，产品推荐占30%-

### 13.5 产品植入自然过渡技巧

1. **问题驱动**：先讲一个普遍问题，产品作为解决方案出现
2. **故事驱动**：讲一个真实案例，案例中自然用到产品
3. **方法驱动**：先教方法，"如果你想更系统地学，可以看XX"
4. **数据驱动**：用效果数据说话，"XX学员3个月后收入涨了X倍"

> 对大多数普通人而言，二三十万算不上巨款，却是一个人生关键拐点。为什么这么说？按正常的逻辑来说，赚到了这笔钱，本该是值得庆幸的事。可我见过不少人，攒到这个数后就开始不对劲。他们非但没有继续往上走，反而开始走下坡路。说到底，很多人始终没意识到：只会赚钱，从来都不等于能真正变有钱。

## 十五、朱雀AI检测规避要点

⚠️ **注意：朱雀检测是写作的最后一步，不是第一步。先写好内容，最后过检。内容质量优先于检测分数。**

基于25轮实测总结的核心规则（详细规则见用户级MEMORY.md）：

1. **事件链驱动**：每段推进一个具体动作/事件，不是在"描述一个人是什么样的人"
2. **句子必须完整**：信息充分，逻辑完整，不是说短就叫口语化。真人不发电报式碎句
3. **人物背景必须交代**：这个人是谁、干什么工作、家里什么情况——真人叙事的标配
4. **不说教**：用场景承载心理，不直白写"心里很矛盾""心里才踏实"
5. **不用过渡句做铺垫**："按说该享福了""说个有意思的""说白了"这种废话删掉——删掉后不影响意思就不该出现
6. **数字散落**：不在一段集中罗列数字，散在不同段落
7. **结尾自然延伸**：不是排比/对仗收束，是自然的延伸

## 十七、深度学习笔记写法详解（来自12篇DanKoe系列）

### 17.1 什么是深度学习笔记

不是写书评/课程评价，而是"我深度学习了XX后，提取出对读者有用的认知/方法"。作者位置是"学习者和实践者"，不是"评价者"。

### 17.2 标题公式

```
[年龄标签]深度学习[人物/课程]才发现：[核心认知/方法]
```

示例：
- "40岁深度学习油管大神才发现：一人公司年入千万，每天只工作4小时"
- "40岁深度学习油管大神才发现：搞钱的这8个关键认知，99%的人都忽略了"
- "40岁深度学习油管大神才发现：3步极简写作法，零基础小白也能日更变现！"

### 17.3 开头公式

```
"最近，我深度学习了XX的XX" → 金句预告(2-3句) → 强调"我"的付出(X天/X遍) → 抛出核心价值
```

关键要素：
- 用2-3个最具冲击力的金句快速勾住读者
- 强调"我"的付出：让读者觉得"我替你省了时间"
- 连续系列用"这是我深度学习XX的第X篇笔记"建立追踪感

### 17.4 主体写法

- 01：是什么——核心概念/认知
- 02：为什么——为什么重要/颠覆
- 03：怎么做——实操步骤+自身实践经历（必须有！）
- 04：写在结尾

**关键规则**：必须有"我是怎么实践的"段落。不是只讲理论，要讲"我怎么1:1复刻的，效果如何"。这才是深度学习笔记区别于普通书评的核心。

### 17.5 素材复用技巧

同一核心素材可以产出2-3篇不同角度：
- 系列连续法：第1篇讲概念→第2篇讲实操→第3篇讲案例
- 深浅搭配法：1篇金句清单体(宽)+1篇深度解读(深)

## 十八、读书观点文写法详解（来自5篇《天道》/《肖申克》系列）

### 18.1 什么是读书观点文

不是写书评，是从经典作品中提取一个和赚钱/成长相关的认知，用书中案例+现实案例+个人经历三重印证。

### 18.2 标题公式

```
[年龄]重读/三刷[书名]才发现，[反常识观点]
```

示例：
- "40岁重读《天道》才发现，想赚钱，一个多余的动作都不要有"
- "40岁三刷《肖申克的救赎》才发现：最狠的3个真相，只有1%的人能看破！"
- "40岁重读《天道》才发现，普通人拿关系当人脉，高手用价值换利益"

### 18.3 开头写法

4种开头模式：
1. **咨询场景切入**："今天1v1咨询的伙伴问我..."
2. **普遍痛点切入**："最近两年，身边人说得最多的话，就是，赚钱太难了！"
3. **反常识观点直击**："任何社交都换不来人脉。真正的人脉，只能用价值交换。"
4. **画面感切入**：用书中经典台词/画面开篇

### 18.4 主体写法

**三重印证法**：
- 书中案例详细拆解（不是一笔带过，是像讲故事一样讲情节）
- 现实案例映射（和读者相关的现实困境）
- 作者亲身经历证明

**对比手法**（高频使用）：
- 成功角色 vs 失败角色（欧阳雪 vs 叶晓明）
- 书中世界 vs 现实世界

### 18.5 金句来源

- 书中台词改编（最常用，最有辨识度）
- 原创金句
- 名人名言引用
- 心理学/经济学概念引入（"时间贴现""体制化""弱势文化"）

### 18.6 结尾特色

- 回扣书中角色/台词
- "你也可以"——给予读者信心
- 引用书中金句收尾

## 十九、访谈金句清单体写法详解（来自9篇访谈系列）

### 19.1 什么是访谈金句清单体

替读者看、帮读者提炼——把X小时的访谈/X万字的文稿，提炼成20-30条金句精华。读者花5分钟就能获取核心价值。

### 19.2 标题公式

```
X小时访谈，X万字文稿，[人物]对谈[人物]，才发现，X个反常识真相（值得收藏）
```

示例：
- "4小时访谈，9万字文稿，罗永浩李想：反常识的29个认知真相（值得收藏）"
- "3小时访谈，4.6万字文稿，三刷鲁豫对谈62岁章小蕙，才发现，人生翻盘的20个反常识真相（值得收藏）"

标题共性：数字+名人背书+反常识+收藏引导

### 19.3 两种形态

**金句型清单体**（碎片化友好）：
- 编号从01开始，每条50-150字
- 总量20-30条，分4-6个板块
- 每条独立可消费，拉到哪里读哪里
- 每条 = 标题句 + 1-3句展开解释
- 适合：信息量极大、金句密度极高的素材

**解读型清单体**（深度友好）：
- 聚焦3个核心观点深度展开
- 每个观点800-1500字
- 有案例有理论有实操
- 适合：一个素材想深度挖掘

### 19.4 开头万能公式

```
热点事件引入 → 金句快速预告(3-5句) → "我"的付出(X天/X遍/X万字) → 价值承诺(X条真相)
```

### 19.5 结尾万能公式

```
排比总结(呼应正文板块) → 引用原话金句 → "你我共勉" → 互动引导
```

### 19.6 素材复用技巧

同一访谈素材可以产出2-3篇不同文章：
- **角度1**：直接写访谈解读（段永平访谈解读）
- **角度2**：对比角度重新包装（王石vs段永平对比）
- **角度3**：金句清单版+深度解读版

## 二十一、五大需求层次框架（来自25讲第07讲）

选题时，用**马斯洛五大需求层次**来判断痛点程度和价值大小。

| 需求层次 | 对应痛点 | 爆文示例 | 紧迫度 |
|---------|---------|---------|---------|
| **生理需求** | 赚钱、省钱、生存 | "普通人赚不到钱，原因就1个！" | ⭐⭐⭐⭐⭐ |
| **安全需求** | 稳定、保障、避险 | "你的存款有二三十万，请记住这3条保富法则" | ⭐⭐⭐⭐⭐ |
| **社交需求** | 认同、归属、面子 | "真正赚钱厉害的人，都有一个共性" | ⭐⭐⭐⭐ |
| **尊重需求** | 成就、地位、认可 | "40岁搞副业，才发现：一个人成大事前最好的状态，销声匿迹！" | ⭐⭐⭐ |
| **自我实现** | 成长、意义、使命 | "赚钱的终极意义，是兑换更高级的人生体验！" | ⭐⭐ |

**使用技巧**：
1. 越底层的需求，受众越广，痛点越痛
2. 一篇文章的选题，最好击中1-2个需求层次
3. 标题可以直接点明需求（如"赚钱""保富""认同"）
4. 生理和安全需求是"一级痛点"，必须解决，不得不看

## 二十二、标题6种关键词（来自25讲第08讲）

标题中使用这6种关键词，可以显著提升打开率。

| 关键词类型 | 作用 | 标题示例 |
|-----------|------|---------|
| **热点** | 借势时效性 | "罗翔3200万粉丝之后，鲁豫挖出了扎心的生存真相" |
| **痛点** | 直击读者困境 | "如果你的存款有二三十万，请牢牢记住这3条" |
| **共鸣** | 替读者说话 | "40岁重读《天道》才发现，想赚钱，一个多余的动作都不要有" |
| **故事** | 引发好奇心 | "体制内裸辞，每天只干4小时，一人公司跑通6种模式" |
| **数字** | 制造具体感 | "一篇文章2300块，从40岁裁员危机，到全网15万粉" |
| **超级符号** | 借用文化认知 | "《天道》""粥左罗""财务自由""纳瓦尔" |

**组合使用**：
- 数字 + 痛点 + 超级符号 = 最强组合
- 示例："40岁重读《天道》才发现，普通人挣钱的路子，从古至今就一个"

## 二十三、改文7大步骤（来自25讲第11讲）

写完初稿后，按以下7步顺序修改，确保文章质量。

### 第1步：检查框架结构

- [ ] 01/02/03分段是否清晰？
- [ ] 每个分段是否只讲一个观点？
- [ ] 分段之间是否有逻辑递进关系？
- [ ] 有没有可以删除的段落（删掉不影响意思）？

### 第2步：全篇通读

- [ ] 从头到尾读一遍，感受节奏
- [ ] 有没有拗口的地方？
- [ ] 有没有逻辑跳跃的地方？
- [ ] 有没有重复表达的地方？

### 第3步：精炼语言

- [ ] 删掉所有过渡废话（"说个有意思的""按说""说白了"）
- [ ] 把长句拆成短句（30字以内）
- [ ] 把碎句合并成完整句子（不要电报式）
- [ ] 删掉所有形容词堆砌，用动词和数字替代

### 第4步：强化观点和价值

- [ ] 每个分论点是否有案例佐证？
- [ ] 每个案例是否有具体数字？
- [ ] 核心观点是否重复了2-3次？
- [ ] 有没有"可能""也许""我觉得"等模糊词？替换为"一定""绝对""千万不要"

### 第5步：增加案例和数据

- [ ] 每个观点是否有至少1个案例？
- [ ] 案例是否真实可查？
- [ ] 数据是否具体（不用"很多""一些"）？
- [ ] 有没有可以增加的个人经历案例？

### 第6步：检查开头结尾呼应

- [ ] 开头抛出的问题，结尾是否回答了？
- [ ] 开头的钩子，结尾是否收回来了？
- [ ] 结尾是否呼应了标题的核心观点？
- [ ] 结尾是否有互动引导？

### 第7步：关注细节

- [ ] 标点符号是否正确？（中文用中文标点）
- [ ] 数字是否一致？（全文统一用阿拉伯数字）
- [ ] 重点句是否加粗？
- [ ] 段落是否短小（1-3行）？
- [ ] 图片位置是否合理（每300-500字配1张）？

**改文心态**：
- 好文章是改出来的，不是写出来的
- 初稿完成后，放一天再改，效果更佳
- 改文时，把自己当成读者，不是作者

## 二十四、100%人工创作文风指南

基于Vicky的25篇100%朱雀人工认证爆文分析得出。

### 24.1 整体文风特征

1. **直接对话式开头**："你好，我是粥左罗。"或"什么样的人，注定赚钱越厉害？答案是：懂的升维的人。"
2. **短文段为主**：每段1-3句话，大量使用换行增强节奏感
3. **数字编号结构**：使用01、02、03...来划分章节
4. **大量使用"你"和"我"**：第二人称"你"+第一人称"我"，直接与读者对话
5. **个人故事和案例**：分享个人失败经历、决策过程、真实数字
6. **简单清晰的语言**：避免复杂词汇和句式，让读者感觉像在听朋友聊天
7. **重复强调关键点**：重要观点重复2-3次，加深读者印象
8. **结尾固定格式**："以上，有启发，请帮忙点赞、转发分享，感谢！"

### 24.2 具体写作技巧

**标题风格**：
- 数字+反常识
- 痛点+解决方案
- 疑问式

**开头技巧**：
- 提问式：直接提出一个让读者思考的问题
- 反常识事实：给出一个颠覆常识的观点
- 个人故事：用一个简短的个人经历引入

**正文结构**：
- 每个编号下有小标题
- 小标题后跟2-3段解释
- 使用例子、故事、数据来支撑观点
- 段落之间用空行分隔

**例子使用**：
- 个人例子：用自己的经历
- 名人例子：用知名人物的故事
- 假设例子："假设你..."、"比如..."
- 对比例子：正面 vs 反面

**语言风格**：
- 使用口语词："其实"、"但是"、"因为"、"所以"
- 避免书面语："然而"、"因此"、"综上所述"
- 使用强调词："一定"、"非常"、"特别"、"真的"
- 使用反问句："为什么呢？"、"你有没有想过？"

### 24.3 避免AI味的关键点

❌ **不要使用**：
- 排比句
- 对仗句
- 复杂句型
- "不是...而是..."句型
- 引号强调词语
- "真正的..."、"你是不是也..."句型

✅ **要使用**：
- 简单的句子结构
- 自然的语言节奏
- 个人化的表达

### 24.4 文风检查清单

写完文章后，用这个清单检查：

1. [ ] 开头是否直接、有对话感？
2. [ ] 段落是否简短（1-3句话）？
3. [ ] 是否大量使用了"你"和"我"？
4. [ ] 是否分享了个人故事或案例？
5. [ ] 语言是否简单清晰、口语化？
6. [ ] 是否有AI味的句型（排比、对仗、复杂句型）？
7. [ ] 结尾是否有呼吁行动？
8. [ ] 整体读起来是否像真人在说话？

### 24.5 模仿练习

要掌握这种文风，可以：
1. 反复阅读25篇100%人工认证文章，体会语言节奏
2. 模仿写作：选择一个主题，按照这种风格写一篇短文
3. 对比检查：将自己写的文章与5篇对比，找出差距
4. 反复修改：直到文章读起来像真人在说话

**重要提醒**：这种文风的核心是把读者当朋友，用聊天的方式分享有价值的内容，而不是用机器人的方式输出信息。
